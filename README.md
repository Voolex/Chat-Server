# Chat-Server
Серверная часть чата

#### Стек технологий
* Java
* Spring Boot
* MySQL
* Hibernate

#### Описание возможностей
* Личные сообщения
* Групповые чаты
* Секретные личные чаты с шифрованием end-to-end


#### Доступные роли
* ROLE_USER - простой user
* ROLE_ADMIN - роль администратора. Функционал пока не придуман.
